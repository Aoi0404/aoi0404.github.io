図形

三角形の相似条件
  • ３組の辺の比が全て等しい。
  • ２組の辺の比とその間の角がそれぞれ等しい。
  • ２組の角がそれぞれ等しい。
相似比と周の比　面積比
  • 相似な平面図形の周の長さの比は相似比に等しい。相似比が m : n →周の n
  • 相似な平面図形の面積比は相似比の２乗に等しい。相似比が m : n →面積比は m2 : n2
相似比と表面積比　体積比
  • 相似な立体の表面積比は相似比の２乗に等しい。相似比が m : n →表面積比は m2 : n2
  • 相似な立体の体積比は相対比の３乗に等しい。相似比が m : n →体積比は m3 : n3
